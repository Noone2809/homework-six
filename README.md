# Travel-Fly - Homework Six

## Project Overview

Travel-Fly is a responsive travel booking website created for a web development course (Homework Six). This version expands on earlier work by adding multiple destination tour pages, improved layout, and updated styling using SCSS.

## Features

- **Fixed Navigation Header** - Easy access to navigation links (Home, About, Tours)
- **Hero Section** - Eye-catching landing section with a call-to-action button
- **Tour Directory** - Page listing all available tours with images and links
- **Individual Tour Pages** - Separate HTML pages for each destination (Israel, U.S.A., Australia, Canada, Egypt, France, Japan, New Zealand, UAE) with details and images
- **Booking Form** - Interactive form for users to submit travel booking requests including:
  - Personal information (name, email)
  - Travel details (country, hotel)
  - Check-in and check-out dates
  - Guest count (adults, children, rooms)
  - Special message field
- **Responsive Design** - Built with SCSS for maintainable and scalable styling, compiling to CSS
- **Footer** - Copyright information and social media links
- **About Page** - Company information and background page

## Pages

- **Home** - Main landing page with hero section, featured tours, and booking form
- **Tours** - Directory of all available tour destinations
- **About** - Company information and background page
- **Destination Pages** - Individual HTML files for each tour

## Technologies Used

- HTML5
- SCSS/Sass (compiled to CSS)
- Responsive Web Design principles

## Development

Styles are written in SCSS and compiled via a watch task (see `package.json` for scripts).
